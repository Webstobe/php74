# php74 (with SSL)

------------

- composer from official image

  - vim
  - git
  - unzip / zip
  - rsync
  - graphicsmagick
  - ghostscript
  - mariadb-client
  - ssh
  - jpegoptim
  - optipng
  - gifsicle
  - poppler-utils (pdftotext / pdftohtml ...)
  - redis for PHP
